# Codegen

CLI for code generation using [Handlebars](https://handlebarsjs.com/guide/) templates and [gray-matter](https://www.npmjs.com/package/gray-matter) to get meta data from a YAML part of a file.

## Install

``` sh
npm install -D github:serhii-untilov/codegen
```

## Create a template

Create a `templates` folder in the root of your project.

Create a template file, like a `templates\python.class\name.snakeCase.py.hbs`

``` hbs
---
name: python.class
description: Python class template
target: "{{ snakeCase name }}.py"
inject:
    mode: create
    overwrite: false
vars:
    - name
    - baseClass
---

# Generated by {{codegen.name}} v{{codegen.version}} at {{date}} {{time}}
# Homepage: {{codegen.homepage}}
# Template: {{template}}

class {{pascalCase name}}:
    def __init__(self):
        pass
```

## Generate code

To generate code run:

``` sh
npx codegen --templates ./templates/python.class --name example --output src
```

The Codegen will generate the file `src\example.py`

## Getting help

``` sh
npx codegen --help
npx codegen --version
```

## Update

``` sh
npm update codegen
```

## Development

``` sh
npm run build
./dist/app.js --templates ./templates/ts.class --output ./out
```
